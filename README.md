# Little_War_Game

<i>
一開始會在雙方各創造一個城堡</br>
A方的城堡等級比較低<br>
但是城堡等級只差在城堡血量<br>
而顯示的value指的是這個士兵在路線上的位置<br>
</i>

</br>
戰鬥中控制指令板</br>
<b>q:</b>離開</br>
<b>a:</b>新增一隻弓箭兵給A</br>
<b>s:</b>新增一支劍兵給A</br>
<b>z:</b>新增一隻弓箭兵給B</br>
<b>x:</b>新增一支劍兵給B</br>
其他任意鍵 下一步</br>
